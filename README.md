# Ultimate-Goal

## Controles Tele Operado

Gamepad 1 [Controle de movimentação]

Dpad
Up: Vira para frente

Left: Vira para esquerda

Right: Vira para direita

Back: Vira para trás

.

Joystick Esquerdo

Y: Anda para frente e trás

X: Anda para esquerda e direita

Y+X: Anda na diagonal

.

Joystick Direito

X: Gira para esquerda e direita

.

Bumper Esquerdo

1 toque: Reduz a velocidade em 25%

.

Bumper Direito

1 toque: Aumenta a velocidade em 25%

.
